Proje 2
[16,21,11,8,12,22] -> Merge Sort

Yukarıdaki dizinin sort türüne göre aşamalarını yazınız.
Big-O gösterimini yazınız.

            [16,21,11,8,12,22]

[16,21,11] 			[8,12,22]

[16] [21,11]                    [8] [12,22]

[10] [21] [11]                  [8] [12] [22]

 [10,11,21]                      [8,12,22]

		[8,10,11,12,21,22]

Big-O gösterimi =>  2^x = n 
		    x = logn --> O(nlogn)
